## üí° Tips for Using Advanced Tools

### When to Use Each Tool:

**During First Draft**:

- AI Writing Assistant (for word choice and completion)
- Beat Sheet Generator (to check structure)

**During Revision**:

- Dialogue Enhancer
- POV Checker
- Clich√© Detector
- Emotion Tracker

**During Final Polish**:

- Readability Metrics
- Motif Tracker
- All tools for comprehensive review

### Best Practices:

1. **Don't Over-Rely on Tools**: Use them as guides, not rules
2. **Check Regularly**: Run analysis after major revisions
3. **Focus on Patterns**: One issue is fine, patterns indicate problems
4. **Know Your Genre**: Different genres have different expectations
5. **Trust Your Voice**: Tools suggest, but you decide

### Performance Tips:

- Tools work best on complete chapters or scenes
- Longer texts provide more accurate analysis
- Some tools may take a few seconds for large manuscripts
- Close unused tool modals to improve performance

## üéì Understanding the Scores

### Readability Scores:

- **90-100**: Very Easy (5th grade)
- **80-90**: Easy (6th grade)
- **70-80**: Fairly Easy (7th grade)
- **60-70**: Standard (8th-9th grade)
- **50-60**: Fairly Difficult (10th-12th grade)
- **30-50**: Difficult (College)
- **0-30**: Very Difficult (College Graduate)

### Consistency Scores:

- **90-100%**: Excellent consistency
- **70-89%**: Good, minor issues
- **50-69%**: Fair, needs attention
- **Below 50%**: Poor, significant issues

### Confidence Scores:

- **80-100%**: High confidence in detection
- **60-79%**: Medium confidence
- **Below 60%**: Low confidence, manual review recommended

## üÜò Troubleshooting

**Tool not opening?**

- Ensure you're in Writer Mode (not Analysis Mode)
- Check that you have text in the editor
- Try refreshing the page

**No results showing?**

- Some tools require minimum text length
- Ensure your text has clear structure (paragraphs, chapters)
- Check filter settings

**Suggestions seem off?**

- Tools use pattern matching and may not understand context perfectly
- Always review suggestions before accepting
- Use your judgment as the author

**Performance issues?**

- Close unused tool modals
- Work on smaller sections for very large manuscripts
- Clear browser cache if needed

# Keyboard Shortcuts Guide

Quick reference for all keyboard shortcuts available in QuillPilot's Writer Mode.

> **Note**: On macOS, use `Cmd` (‚åò) instead of `Ctrl`. On Windows/Linux, use `Ctrl`.

## Text Formatting

| Shortcut       | Action           | Description               |
| -------------- | ---------------- | ------------------------- |
| `Ctrl/Cmd + B` | **Bold**         | Make selected text bold   |
| `Ctrl/Cmd + I` | _Italic_         | Make selected text italic |
| `Ctrl/Cmd + U` | <u>Underline</u> | Underline selected text   |

## Headings & Block Formatting

| Shortcut               | Action    | Description                      |
| ---------------------- | --------- | -------------------------------- |
| `Ctrl/Cmd + Alt + 1`   | Heading 1 | Convert to Heading 1 (largest)   |
| `Ctrl/Cmd + Alt + 2`   | Heading 2 | Convert to Heading 2             |
| `Ctrl/Cmd + Alt + 3`   | Heading 3 | Convert to Heading 3             |
| `Ctrl/Cmd + Alt + 4`   | Heading 4 | Convert to Heading 4             |
| `Ctrl/Cmd + Alt + 5`   | Heading 5 | Convert to Heading 5             |
| `Ctrl/Cmd + Alt + 6`   | Heading 6 | Convert to Heading 6 (smallest)  |
| `Ctrl/Cmd + Alt + 0`   | Paragraph | Convert back to normal paragraph |
| `Ctrl/Cmd + Shift + .` | Quote     | Convert to blockquote            |

> **Note**: Keyboard shortcuts are also displayed in the Block Type dropdown menu for quick reference.

## Document Control

| Shortcut               | Action         | Description                              |
| ---------------------- | -------------- | ---------------------------------------- |
| `Ctrl/Cmd + S`         | Save           | Save document (auto-save is also active) |
| `Ctrl/Cmd + Z`         | Undo           | Undo last action                         |
| `Ctrl/Cmd + Shift + Z` | Redo           | Redo previously undone action            |
| `Ctrl/Cmd + Y`         | Redo           | Redo (Windows/Linux alternative)         |
| `Ctrl/Cmd + F`         | Find & Replace | Open find and replace dialog             |
| `Ctrl/Cmd + K`         | Insert Link    | Add hyperlink to selected text           |

## Editor Navigation

| Shortcut          | Action         | Description                          |
| ----------------- | -------------- | ------------------------------------ |
| `Tab`             | Indent         | Insert 4 spaces for indentation      |
| `Shift + Tab`     | Outdent        | Reduce indentation (browser default) |
| `Home`            | Line Start     | Move to beginning of line            |
| `End`             | Line End       | Move to end of line                  |
| `Ctrl/Cmd + Home` | Document Start | Jump to beginning of document        |
| `Ctrl/Cmd + End`  | Document End   | Jump to end of document              |

## Column Layout Shortcuts

| Shortcut                   | Action        | Description                                |
| -------------------------- | ------------- | ------------------------------------------ |
| `Ctrl/Cmd + A` (in column) | Select Column | Selects only content within current column |

> **Note**: When cursor is inside a column, Cmd/Ctrl+A selects only that column's content, not the entire document.

## Pro Tips

### Quick Heading Creation

1. Type your text
2. Select it (or just place cursor on the line)
3. Press `Ctrl/Cmd + Alt + [1-6]` to convert to heading
4. Press `Ctrl/Cmd + Alt + 0` to convert back to paragraph

### Working with Columns

1. Insert columns using the ‚ò∞ button in the right toolbar
2. Click inside any column to edit
3. Use Cmd/Ctrl+A to select all content within just that column
4. Drag from the handle bar at top to reposition
5. Right-click the handle for options (show/hide, delete)

### Fast Formatting Workflow

1. Write without formatting first
2. Select text you want to format
3. Use shortcuts to apply bold, italic, etc.
4. Continue writing

### Efficient Navigation

- Use `Ctrl/Cmd + F` to quickly find text
- Use keyboard shortcuts to format as you go
- Enable **Typewriter Mode** (‚å®Ô∏è button) to keep cursor centered
